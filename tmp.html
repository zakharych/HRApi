Тематика конференции

  • NETWORK

    • Linux kernel network stack (eBPF, core protocols, device drivers, routing, TC, Netfilter, XFRM, etc.)
    • Linux networking APIs for user-space (best practices, limitations, performance, scalability, etc.)
    • High-performance networking in Linux (XDP, DPDK, hardware offloads)
    • Networking for VMs and containers

  • OBSERVABILITY AND PERFORMANCE ANALYSIS

    • eBPF, BCC and friends
    • ftrace, perf, and tools
    • Linux debugging and troubleshooting
    • Linux performance tuning

  • SERVER AND STORAGE

    • Support for new hardware and protocols (SMR, NVMe and NVMe-oF, LightNVM/OpenChannel SSD, etc.)
    • Persistent memory in Linux (NVDIMMs, DAX, PM-aware filesystems, etc.)
    • Linux filesystems and distributed storage (btrfs, xfs, NFS, SMB, Ceph, GlusterFS, etc.)
    • Linux memory management
    • Storage performance and scalability
    • Open-source databases

  • CLOUD

    • Open source cloud stacks (OpenStack, CloudStack, oVirt, etc)
    • Open source SDN/SDS (OpenvSwitch, GlusterFS, etc)

  • VIRTUALIZATION

    • QEMU/KVM high performance I/O (virtio, vhost, data-planes)
    • Virtualization management (libvirt and project built on top of it)
    • General virtualization topics (nested virtualization, live migration, QoS)
    • Containers under the covers (cgroups, namespaces, capabilities)
    • Container runtimes, networking, storage, and security
    • Container M&O (K8s, Docker, etc

  • EMBEDDED & MOBILE

    • Linux-based electronic systems development
    • Open source mobile operating systems
    • Kernel and system software tuned for embedded application
